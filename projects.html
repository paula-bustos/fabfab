SOME SKETCHES

MECHANISM

We decided to create a mechanism which would have two parts, a negative and positive which could be inserted into each other. By designing a "key" we are able to join them together.

We designed the set of cubes and other polygons with parametric design in order to easily change the dimensions according to the needs of every project. Every side of the cube can be personalized in case you need more assembly holes or just a plain face. The sides of the shapes can be fitted into other shapes, like the square can be a side of the triangle shape.

We would also like to explore how these modules could work with electronics so we decided to leave some holes in order for cables to pass through. As for now we are going to experiment with some led lights. The material we chose is color metacrilate which we think will give the furniture piece a more modern and fun appearance. This will also allow the inside mechanism to be visible and the connections of future electronics.
